This chart shows how the total amount of kWh used today for each usage category. A Watt hour (Wh) is a unit of energy, which tells us about the amount of work that has been done by some devices. A kWh or kilo Watt hour is just 1000 Watt hours. Watt hours refer to enegy used / work done for some time period. For example: a 60 Watt lightbulb that is on for 1 hour will consume 1 Wh of energy.
